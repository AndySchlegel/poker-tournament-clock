# Poker Tournament Clock

Premium browser-based poker tournament clock for home games and live TV/monitor display.

## Features

- Large fullscreen tournament timer
- Configurable blind levels, ante, level duration and breaks
- Player count, players left, buy-ins, rebuys, add-ons
- Chips in play, average stack and prize-pool estimates
- Admin panel with import/export JSON
- LocalStorage persistence
- Presets for Turbo and Deepstack structures
- Vitest-covered clock/structure engine

## Development

```bash
npm install
npm run dev
```

## Verify
